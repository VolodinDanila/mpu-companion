# MPUCompanion

умный помощник студента московского политехнического университета

## о проекте

приложение для организации студенческой жизни:
- управление расписанием занятий
- планирование маршрутов до корпусов
- напоминания о встречах и мероприятиях
- учёт времени на дорогу и утреннюю рутину

## технологии

- react native + expo
- react navigation
- asyncstorage для локального хранения

## запуск

```bash
npm install
npm start
```

для запуска на устройстве:
```bash
npm run android  # android
npm run ios      # ios
npm run web      # веб-версия
```
